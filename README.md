## Functionality of the application

The application helps you can sell goods and services from anywhere in the world. This application has 2 parts. In the Admin application are: Users, Categories, Brands, Products, Customers, Shipping, Orders, Sales Report, Reviews, Questions, and Settings. In the Shopping application are: categories listing, product details, search products, shopping cart, checkout, manage orders, tracking orders, and customer registration.

## üöÄ Usage

Make sure you have:
* [ Setup Java Development Kit (JDK) ](https://www.oracle.com/tr/java/technologies/downloads/) 
*   Setup IntelliJ IDEA  
* [ Setup MySql Community Server ](https://dev.mysql.com/downloads/mysql/)
* Install Git and Maven

Certainly, you can use Spring Tool Suite as an alternative to IntelliJ IDEA

Just run the project in IntelliJ and use this link http://localhost:8080/ShopmeAdmin/  to access the Web application.
