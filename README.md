<h1 align="center">🛍️ Welcome to E-commerce Website 👋</h1>

**E-commerce Shopme** is a full-stack web application that allows you to sell goods and services worldwide.  
It includes both **Admin** and **Shopping** modules with complete e-commerce functionality.  

---

## ✨ Features

### 🛠️ Admin Application
- Manage **Users, Categories, Brands, Products, Customers, Shipping, Orders**
- Generate **Sales Reports**
- Handle **Reviews, Questions, and Settings**
- Export data to **CSV, Excel, and PDF**
- Pagination and sorting for large datasets
- Secure access based on user roles (Admin vs. non-Admin)

### 🛒 Shopping Application
- Browse product categories  
- View detailed product information  
- Search products by keywords  
- Add products to shopping cart  
- Checkout and manage orders  
- Track order status  
- Customer registration and authentication

## 🚀 Getting Started

### Prerequisites
Make sure you have installed:
- [Java Development Kit (JDK)](https://www.oracle.com/tr/java/technologies/downloads/)  
- [IntelliJ IDEA](https://www.jetbrains.com/idea/) (or Spring Tool Suite as alternative)  
- [MySQL Community Server](https://dev.mysql.com/downloads/mysql/)  
- [Git](https://git-scm.com/) and [Maven](https://maven.apache.org/)  

### Run the Project
1. Clone the repo:
   ```bash
   git clone https://github.com/TxmMinh/Spring-Boot-E-commerce-Shopme.git
   cd Spring-Boot-E-commerce-Shopme
2. Import into IntelliJ IDEA (or STS).
3. Configure the MySQL database connection.
4. Run the project.
5. Access the Admin application at:
👉 http://localhost:8080/ShopmeAdmin/
